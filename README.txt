
Module: Mobile Link Handler mobile_link_handler
Author: Ken Hawkins <http://AllAboutKen.com>


Description
===========
A simple set of JS detects that:
- Ensure iOS (and other mobile browsers) don't open links in external tabs
- Ensure Facebook links try to open in the native FB App

Requirements
============

* JS.


Installation
============
* Copy the 'mobile_link_handler' module directory in to your Drupal
sites/all/modules directory as usual.


Usage
=====
Effect to page loads is automatic, just enable.
